On the ubuntu, just cp the font files into /usr/share/fonts/truetype
then the emacs configration file can found the font

第一步：解压字体到任意目录下。
第二步：以root权限执行文件管理器：
   1. sudo Nautilus
复制代码
如果是xubuntu，则执行
   1. sudo thunar
复制代码
第三步：把字体文件复制到/usr/share/fonts/truetape/目录下
第四步：刷新字体缓存信息：
   1. sudo fc-cache
复制代码
第五步：重启，然后设置字体：系统－首选项－外观－字体（xubuntu对应的是用户界面首选项），全部设置成你刚安装的字体。
